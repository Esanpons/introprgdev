#########
Git bàsic
#########

A aquesta secció hi trobaràs com realitzar les tasques que més
freqüentment necessitaràs de Git.

Et proposo que segueixis aquesta secció com un tutorial pràctic.

Comença creant un directori i hi afegeixis algun fitxer. Per exemple:

.. code-block:: console

    $ mkdir projecte
    $ cd projecte
    projecte$ echo "hola" > hola.txt
