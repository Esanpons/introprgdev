###############################
Afegir fitxers a la gestió: add
###############################

Un cop has creat el repositori Git, voldràs afegir-hi fitxers. Potser
són fitxers nous o potser d'existents. Sigui com sigui, quan tinguis
fitxers per a accedir-hi, faràs servir la comanda ``git add``

.. code-block:: console

    projecte$ git add hola.txt

Amb això simplement hem indicat a Git que la propera vegada que
registrem els canvis (amb ``commit``) comenci a considerar aquest
fitxer com a gestionat.

Podem fer que Git afegeixi el contingut de tot un directori. Per
exemple, la següent comanda afegirà tots els codis font java del
directori src

.. code-block:: console

    $ git add src/*.java

I la següent comanda afegirà tot el contingut de la carpeta actual

.. code-block:: console

    $ git add .

Una altra opció interessant d'aquesta comanda és afegir **tots** els fitxers no
gestionats

.. code-block:: console

    $ git add --all
