.. rubric:: Enunciat

En aquest subexercici només hauràs de preocupar-te per desenvolupar el
codi del programa principal de manera que sigui capaç d'oferir les
funcionalitats bàsiques d'ajuda i finalització.

Això inclou les següents *comandes*:

* ``quit``: finalitza l'execució

* ``help``: mostrarà un missatge d'ajuda per poder saber quines són les
  comandes disponibles.

Quan la comanda no és una de les anteriors, s'informa amb el missatge ``ERROR:
comanda no reconeguda. Escriviu help per ajuda``.

Una simulació d'execució del programa podria ser la següent:

.. code-block:: console
    :emphasize-lines: 2, 4, 6-12, 15

    $ java Puzle 
    Gestor de puzles d'escacs. Escriviu help per ajuda.
    puzle> ajuda
    ERROR: comanda no reconeguda. Escriviu help per ajuda.
    puzle> help
    Comandes disponibles:
    help
    cp «nom1» «nom2»
    add «nom» «figura» «fila» «columna»
    del «nom» «fila» «columna»
    mv »nom» «fila0» «columna0» «fila1» «columna1»
    new «nom»
    quit
    puzle> quit
    adèu

Nota: fixa't que el *prompt* no salta de línia. Per aquesta raó apareix
seguit de la comanda introduïda pels usuaris.

En cas que et calgui, considera la següent plantilla

.. code-block:: java

    /* XXX falta capçalera */
    public class Puzle {
        public static void main(String[] args) {
            mostraBenvinguda();
            while (true) {
                mostraPrompt();
                String comanda = Entrada.readLine().strip();
                if (comanda.isEmpty()) continue;
                if (comanda.equals("quit")) break;
                if (comanda.equals("help")) {
                    mostraAjuda();
                    continue;
                }
                mostraErrorComandaDesconeguda();
            }
            mostraComiat();
        }
        // XXX falten els mòduls que són cridats per aquest main()
    }

Per poder passar les proves, et caldrà implementar, com a mínim, el
mòdul ``mostraAjuda()``, que, sense paràmetres, mostra el text d'ajuda.

.. note:: Les cometes franceses ``'«'`` i ``'»'`` no solen estar mapejades a
    tots els teclats. Si és el teu cas, et proposo que simplement facis un
    *copy&paste* de les d'aquest enunciat.
