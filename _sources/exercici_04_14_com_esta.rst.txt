.. rubric:: Enunciat

Afegeix al repertori de mètodes del Renat els següents:

.. uml::
    :align: center

     @startuml
     hide circle
     skinparam monochrome true
     skinparam classAttributeIconSize 0

     class GatRenat {
         + estaViu(): boolean
         + estaDret(): boolean
         + estaAssegut(): boolean
         + estaEstirat(): boolean
     }
     @enduml

1. ``boolean estaViu()`` // cert si encara li queda alguna vida

2. ``boolean estaDret()``, ``boolean estaAssegut()``, ``boolean estaEstirat()``

Completa la plantilla següent:

.. code-block:: java
    :linenos:

    public class UsaGatRenat {
        public static void main(String[] args) {
            GatRenat renat = new GatRenat();
            System.out.println("Inicialment renat.estaViu(): " + renat.XXX);
            System.out.println("Inicialment renat.estaDret(): " + renat.XXX);
            System.out.println("Inicialment renat.estaAssegut(): " + renat.XXX);
            System.out.println("Inicialment renat.estaEstirat(): " + renat.XXX);
            System.out.println("Introdueix quantes vides:");
            renat.XXX(Integer.parseInt(Entrada.readLine()));
            System.out.println("Introdueix nova posició:");
            renat.XXX(Entrada.readLine());
            System.out.println("Finalment renat.estaViu(): " + renat.XXX);
            System.out.println("Finalment renat.estaDret(): " + renat.XXX);
            System.out.println("Finalment renat.estaAssegut(): " + renat.XXX);
            System.out.println("Finalment renat.estaEstirat(): " + renat.XXX);
        }
    }

La sortida haurà correspondre a aquesta simulació:

.. code-block:: console
    :emphasize-lines: 2-6, 8, 10-

    $ java UsaGatRenat 
    Inicialment renat.estaViu(): true
    Inicialment renat.estaDret(): false
    Inicialment renat.estaAssegut(): false
    Inicialment renat.estaEstirat(): true
    Introdueix quantes vides:
    0
    Introdueix nova posició:
    dret
    Finalment renat.estaViu(): false
    Finalment renat.estaDret(): true
    Finalment renat.estaAssegut(): false
    Finalment renat.estaEstirat(): false

D'acord, no és massa creïble que el Renat pugui estar a l'hora dret i
mort. Ho deixarem així de moment en favor de la simplicitat. Se t'acut,
però, com podríem arreglar aquesta situació?
