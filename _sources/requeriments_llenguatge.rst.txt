#########################
Llenguatge de programació
#########################

En aquest curs farem servir el llenguatge de programació Java. En concret,
farem servir una implementació anomenada *OpenJDK* en la seva versió 11.

Si tens qualsevol altra implementació o/i versió de Java ja instal·lada,
sempre que la versió sigui >=1.11) no hauries de tenir cap problema per
seguir aquesta part del curs. En cas que la versió sigui anterior, molt
probablement et trobaràs que no podràs realitzar algunes activitats amb
normalitat.

Comprova la teva versió amb les comandes:

.. code-block:: console
    :emphasize-lines: 2, 4-

    $ javac -version
    javac 11.0.12
    $ java -version
    openjdk version "11.0.12" 2021-07-20
    OpenJDK Runtime Environment (build 11.0.12+7-post-Debian-2deb10u1)
    OpenJDK 64-Bit Server VM (build 11.0.12+7-post-Debian-2deb10u1, mixed mode, sharing)

El teu resultat no té perquè ser idèntic als meus. Únicament t'has
d'assegurar que ha trobat la comanda i que la versió és 1.11 o més.

Si no tens instaŀlat Java, et generarà un missatge similar a:

.. code-block:: console
    :emphasize-lines: 2

    $ javac -version
    bash: javac: no s'ha trobat l'ordre

En cas que no tinguis el Java instal·lat, ho pots fer fàcilment amb la
següent comanda que necessitarà privilegis d'administració. Si no els
tens, demana ajuda doncs hi ha alternatives.

.. code-block:: console

    $ sudo apt install default-jdk

Et recomano que no t'obsessionis gaire amb la versió de Java. La darrera
política dels seus propietaris (Oracle) és la d'oferir actualitzacions
freqüents que normalment incorporen millores en elements avançats del
llenguatge.

Una última observació. És important que tinguis clar que aquest **no** és
un curs de Java. Només el fem servir com a exemple per aprendre els
conceptes de programació que després faràs servir en
ves-a-saber-tu-quin-llenguatge (potser ni existeix encara!). Les
característiques particulars de Java fan que sigui una bona elecció per
aquest curs, no pateixis.
