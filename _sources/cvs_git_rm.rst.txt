################
Eliminar fitxers
################

De vegades ens interessar√† eliminar fitxers dels nostres projectes.

Podem fer-ho simplement amb les utilitats del nostre sistema operatiu.
Per exemple:

.. code-block:: console

    projecte$ rm hola.txt
    projecte$ git status
    # On branch master
    # Changes not staged for commit:
    #   (use "git add/rm <file>..." to update what will be committed)
    #   (use "git checkout -- <file>..." to discard changes in working directory)
    #
    #	deleted:    hola.txt
    #
    no changes added to commit (use "git add" and/or "git commit -a")
    projecte$ git commit -am "Eliminat hola.txt"
    [master 4923494] Eliminat hola.txt
    1 file changed, 2 deletions(-)
    delete mode 100644 hola.txt
    $ git status
    # On branch master
    nothing to commit (working directory clean)

Consulta `git rm
<https://www.kernel.org/pub/software/scm/git/docs/git-rm.html>`_ per
veure altres opcions d'esborrar que ens ofereix Git.
