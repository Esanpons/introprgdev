.. rubric:: Enunciat

Desenvolupa una nova versió de l'exercici que :doc:`analitzava un caràcter <exercici_01_66_analisi_caracter>`. 

La nova versió passarà les mateixes proves que l'original però el mòdul
``main()`` se n'encarregarà d'obtenir les dades d'entrada, comprovarà que
siguin vàlides, mostrarà els errors corresponents, i n'extraurà el
caràcter que finalment s'haurà d'analitzar.  Aquest caràcter li passarà al
mòdul ``analitzaCaracter()`` que serà qui se n'encarregui de realitzar
l'anàlisi.

La signatura del nou mòdul serà:

.. code-block:: java

    public static void analitzaCaracter(char caracter)
