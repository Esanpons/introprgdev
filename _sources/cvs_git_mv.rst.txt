##################
Reanomenar fitxers
##################

De vegades ens pot interessar canviar el nom d'un fitxer. Git ens
ofereix aquesta possibilitat fent servir:

.. code-block:: console

    projecte$ git mv hola.txt adeu.txt

Amb això obtenim quelcom equivalent a si fem directament

.. code-block:: console

    projecte$ mv hola.txt adeu.txt
    projecte$ git add adeu.txt

Recorda que Git considera canvis en el contingut de fitxers.
Meta-informació com ara el nom del fitxer o el subdirectori on es
troba queda guardada de manera independent.
