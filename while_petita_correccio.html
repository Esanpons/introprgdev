Petita correcció¶

Ara ja sabem fer el programa corresponent depenent del nombre de valors que hàgim de demanar.

Si ens fixem, però, això no és ben bé el que ens demanava l’enunciat del problema original. Tal i com l’estem resolent, el programa demana sempre tots els valors. És a dir, no s’atura quan llegeix el primer negatiu!

El que voldríem és quelcom similar al següent diagrama:

És a dir, en cas que el primer número llegit sigui negatiu, ja no demanarem el segon!

Això encara ho podem fer amb els coneixements que tenim! Com?

Fàcil, posem el bloc de processament del segon valor dins del primer condicional!

public class SumaPositius {
    public static void main(String[] args) {

        // creem la variable on guardarem el resultat de sumar
        int suma = 0;   // inicialment no hem sumat res i per tant és 0

        // declarem la variable que contindrà els valors llegits
        int valor;

        // processem el primer valor
        System.out.println("Introdueix un valor");
        valor = Integer.parseInt(Entrada.readLine());
        if (valor >= 0) {
            suma = suma + valor;

            // processem el segon valor
            System.out.println("Introdueix un valor");
            valor = Integer.parseInt(Entrada.readLine());
            if (valor >= 0) {
                suma = suma + valor;
            }
        }
        // mostrem el resultat
        System.out.println("La suma és " + suma);
    }
}

Fixa’t com, ara només llegim el segon valor si hem sumat el primer, és a dir, si el primer no era negatiu!
