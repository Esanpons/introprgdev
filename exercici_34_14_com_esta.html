Exercici 34_14. Com està Renat?

Context

  • Carpeta de lliurament: 34_14_com_esta/

  • Continguts relacionats: Preguntes a un objecte

  • Com lliurar-lo: instruccions

  • [✓] Exercici amb autoavaluació

Enunciat

Afegeix al repertori de mètodes del Renat els següents:

  1. boolean estaViu() // cert si encara li queda alguna vida

  2. boolean estaDret(), boolean estaAssegut(), boolean estaEstirat()

Completa la plantilla següent:

 1public class UsaGatRenat {
 2    public static void main(String[] args) {
 3        GatRenat renat = new GatRenat();
 4        System.out.println("Inicialment renat.estaViu(): " + renat.XXX);
 5        System.out.println("Inicialment renat.estaDret(): " + renat.XXX);
 6        System.out.println("Inicialment renat.estaAssegut(): " + renat.XXX);
 7        System.out.println("Inicialment renat.estaEstirat(): " + renat.XXX);
 8        System.out.println("Introdueix quantes vides:");
 9        renat.XXX(Integer.parseInt(Entrada.readLine()));
10        System.out.println("Introdueix nova posició:");
11        renat.XXX(Entrada.readLine());
12        System.out.println("Finalment renat.estaViu(): " + renat.XXX);
13        System.out.println("Finalment renat.estaDret(): " + renat.XXX);
14        System.out.println("Finalment renat.estaAssegut(): " + renat.XXX);
15        System.out.println("Finalment renat.estaEstirat(): " + renat.XXX);
16    }
17}

La sortida haurà correspondre a aquesta simulació:

$ java UsaGatRenat
Inicialment renat.estaViu(): true
Inicialment renat.estaDret(): false
Inicialment renat.estaAssegut(): false
Inicialment renat.estaEstirat(): true
Introdueix quantes vides:
0
Introdueix nova posició:
dret
Finalment renat.estaViu(): false
Finalment renat.estaDret(): true
Finalment renat.estaAssegut(): false
Finalment renat.estaEstirat(): false

D'acord, no és massa creïble que el Renat pugui estar a l'hora dret i mort. Ho deixarem així de moment en favor de la simplicitat. Se t'acut, però, com podríem arreglar aquesta situació?
